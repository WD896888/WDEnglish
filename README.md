# 英语练习题网页程序

## 项目概述
这是一个英语练习题网页程序，提供了多种类型的英语练习功能，包括连词成句、语法填空、英文翻译、英语短文和英语作文等模块。目前实现了"连词成句"功能，其他功能正在开发中。

## 功能特点
- **连词成句**：用户可以拖拽单词进行重新排序，然后点击检查按钮验证答案
- **交互式体验**：支持拖拽操作，直观易用
- **即时反馈**：提供正确/错误提示，并显示标准答案
- **丰富题库**：包含120道不同难度的连词成句题目
- **响应式设计**：适配不同屏幕尺寸

## 文件结构
```
/workspace/
├── index.html     # 主页面
├── styles.css     # 样式文件
├── script.js      # JavaScript逻辑和题库
└── README.md      # 项目说明文档
```

## 技术实现
- HTML5 + CSS3 + JavaScript
- 原生拖拽API实现单词重排功能
- 数组随机算法确保每次题目呈现顺序不同
- 字符串标准化算法用于答案比对

## 使用方法
1. 打开 `index.html` 文件即可使用
2. 在"连词成句"模块中，将打乱的单词拖拽到合适位置
3. 点击"检查答案"按钮验证结果
4. 点击"下一题"切换到新题目
